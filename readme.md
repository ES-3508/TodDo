# Node.js API

This is a simple Node.js API for a ToDo application.

## Prerequisites

- Node.js installed on your machine. You can download it from [here](https://nodejs.org/).

## Installation

```bash
$ npm install
```

## Running the app

```bash
# development
$ npm run start
   node index.js

## Test

```bash
# unit tests
$ npm run test
